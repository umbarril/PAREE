# PAREE
Um frontend alternativo para o SIGAA (Sistema Integrado de Gestão de Atividades Acadêmicas) da Universidade Federal da Paraíba.

> NOTA: Este projeto está em desenvolvimento, e não está num estado usável por usuários comuns!

## Introdução

Olá! Este projeto é parte do meu TCC do meu curso de Licenciatura em Ciência da Computação.

O objetivo do projeto é ter uma interface (frontend) alternativa para o SIGAA que qualquer um poderá usar como se fosse a original. Ao mesmo tempo que serve uma proposta de como parte do sistema original poderia ser.

Nisso, o projeto é focado apenas no **Portal Discente**, pois é a parte que mais me interessa pessoalmente e também ao maior número de usuários. O SIGAA é enorme, então seria pretencioso demais almejar implementar tudo o que ele é capaz num TCC.
